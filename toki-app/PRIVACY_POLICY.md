# Politique de Confidentialité - FeedToki

**Dernière mise à jour :** 27 janvier 2025

## 1. Introduction

FeedToki ("nous", "notre", "l'application") respecte votre vie privée et s'engage à protéger vos données personnelles. Cette politique de confidentialité explique comment nous collectons, utilisons, stockons et protégeons vos informations lorsque vous utilisez notre application.

## 2. Données Collectées

### 2.1 Données Personnelles

Nous collectons les informations suivantes :

- **Informations de compte :** Email, nom d'affichage (optionnel)
- **Données de profil :** Poids, objectifs de poids, niveau d'activité
- **Données nutritionnelles :** Repas consommés, aliments, quantités, valeurs nutritionnelles
- **Données de suivi :** Points quotidiens, streaks, évolution du dragon
- **Données techniques :** Type d'appareil, système d'exploitation, navigateur (via analytics)

### 2.2 Données Non Collectées

Nous **ne collectons pas** :
- Votre position géographique
- Vos contacts
- Vos photos (sauf si vous les téléchargez volontairement)
- Vos paiements (si applicable dans le futur, via processeur tiers sécurisé)

## 3. Utilisation des Données

Nous utilisons vos données pour :

1. **Fonctionnement de l'application :**
   - Calculer vos besoins nutritionnels (TDEE)
   - Suivre vos repas et points
   - Afficher vos statistiques et progrès
   - Synchroniser vos données entre appareils

2. **Amélioration du service :**
   - Analyser l'utilisation de l'application (anonymisé)
   - Corriger les bugs
   - Développer de nouvelles fonctionnalités

3. **Communication :**
   - Vous informer des mises à jour importantes
   - Répondre à vos demandes de support

## 4. Partage des Données

### 4.1 Services Tiers

Nous utilisons les services suivants qui peuvent recevoir vos données :

- **Firebase (Google) :** Stockage et synchronisation de vos données
  - Authentification
  - Firestore (base de données)
  - Analytics
  - Voir [Politique de confidentialité Google](https://policies.google.com/privacy)

- **OpenAI :** Analyse des descriptions de repas (si vous utilisez la fonctionnalité IA)
  - Seulement les descriptions de repas (texte)
  - Aucune donnée personnelle identifiante
  - Voir [Politique de confidentialité OpenAI](https://openai.com/policies/privacy-policy)

- **Sentry :** Monitoring d'erreurs (optionnel)
  - Données techniques et logs d'erreurs seulement
  - Voir [Politique de confidentialité Sentry](https://sentry.io/privacy/)

### 4.2 Partage avec D'autres Utilisateurs

- **Base de données alimentaire globale :** Les aliments personnalisés que vous ajoutez peuvent être partagés avec tous les utilisateurs (nom et valeurs nutritionnelles seulement, pas vos repas personnels)

## 5. Sécurité des Données

Nous mettons en place des mesures de sécurité pour protéger vos données :

- Chiffrement des données en transit (HTTPS)
- Authentification sécurisée (Firebase Auth)
- Règles de sécurité Firestore (accès limité à vos propres données)
- Stockage sécurisé côté serveur (Google Cloud)

## 6. Vos Droits (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous avez le droit de :

1. **Accéder à vos données :** Demander une copie de toutes vos données
2. **Corriger vos données :** Modifier vos informations via l'application
3. **Supprimer vos données :** Demander la suppression de votre compte et toutes vos données
4. **Portabilité :** Exporter vos données dans un format standard
5. **Opposition :** Vous opposer au traitement de certaines données

Pour exercer ces droits, contactez-nous à : [votre-email@exemple.com]

## 7. Conservation des Données

- Vos données sont conservées tant que votre compte est actif
- Après suppression de compte, vos données sont supprimées dans les 30 jours
- Les logs techniques peuvent être conservés jusqu'à 90 jours pour le debugging

## 8. Cookies et Technologies Similaires

L'application utilise des technologies similaires aux cookies pour :
- Authentification (session)
- Analytics (Firebase Analytics)
- Aucun cookie de tracking tiers pour la publicité

## 9. Modifications de cette Politique

Nous pouvons mettre à jour cette politique de confidentialité. Nous vous informerons des changements importants via l'application ou par email.

Pour toute question concernant cette politique de confidentialité ou vos données, contactez-nous :
